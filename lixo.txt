# Biblioteca BSI

Projeto Java com Ant para gerenciamento de biblioteca.

---

## Descrição

Este projeto permite:
- Adicionar livros
- Buscar livros
- Remover livros
- Organizar informações da biblioteca

É desenvolvido em **Java 17** e utiliza **NetBeans + Ant**.

---

## Estrutura do projeto

- `src/` → Código-fonte Java
- `nbproject/` → Configuração do projeto NetBeans
- `build/` → Pasta de compilação (ignorada pelo Git)
- `dist/` → Pasta de distribuição (ignorada pelo Git)
- `.gitignore` → Arquivos/pastas ignoradas pelo Git

---

## Como abrir no NetBeans

1. Abra o NetBeans
2. Vá em **Team → Git → Clone**
   - Cole a URL do repositório:  
     ```
     https://github.com/DevUniverse2024/biblioteca-BSI.git
     ```
3. Escolha a **branch principal** (`main`) e a pasta local para clonar
4. Após clonar, vá em **File → Open Project**
5. Selecione a pasta clonada `biblioteca-BSI`  
6. NetBeans deve reconhecer como **Java with Ant project**  
7. Clique **Open Project**

---

## Como rodar o projeto

1. Clique no projeto → **Build** (compila o código)
2. Clique em **Run** (executa o projeto)
3. Verifique se o Java correto está selecionado (Java 17)

---

## Boas práticas ao usar Git

- Antes de começar a programar, sempre faça:  
  **Team → Git → Pull** → atualiza a versão local
- Para enviar alterações:  
  **Team → Git → Commit** → escreva a mensagem → **Push**
- **Não alterar** a pasta `nbproject/private/` (configuração privada do NetBeans)
- Não versionar pastas **build/** ou **dist/** (já estão no `.gitignore`)
